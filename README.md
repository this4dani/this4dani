### What I'm Building

**[Yarn Craft Terminology Database](https://github.com/this4dani/yarn-craft-terminology-database)** -- The first comprehensive, open, API-accessible database of yarn craft terminology. Starting from a crochet foundation with 265+ terms and expanding across all fiber arts disciplines — knitting, weaving, embroidery, and more. Structured data with US/UK translations, categories, quiz system, and educational tools.

- 265+ terms with rich data (difficulty, tags, learning time, best use cases)
- US & UK terminology translations + community slang
- Quiz system with 50+ auto-generated questions
- Free public API served via GitHub CDN
- Community-driven data maintained through Google Sheets pipeline

**Church Website Development** -- Platform-agnostic website builds for churches of all sizes, with a focus on training teams for independence rather than dependency. Currently offering services through [this4dani.com](https://this4dani.com).
